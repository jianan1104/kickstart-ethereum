## Getting Started


### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/jianan1104/kickstart-ethernum.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start next server
   ```sh
   npm run dev
   ```
